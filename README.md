# Home Health Intelligence SaaS Platform

AI-powered compliance, operations, and revenue intelligence system for home health agencies.

## Features
- AI audit engine
- File intelligence analysis
- Consultant white-label dashboard
- Stripe Connect split payments
- Earnings tracking
- Client timeline system

## Tech Stack
- FastAPI
- SQLite / Postgres
- Stripe API
- OpenAI API

## Run Locally
python -m uvicorn app.main:app --reload
